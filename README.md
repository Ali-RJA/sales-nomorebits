# nomorebits - AI Phone Assistant Landing Page

Landing page for nomorebits AI phone assistant service.

## Tech Stack

- Next.js 16
- React 18
- Material UI 5
- Framer Motion
- Phosphor Icons

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Production Build

```bash
npm run build
npm start
```

## Deployment

This project is configured for Railway deployment. Simply connect your GitHub repo and Railway will auto-detect and deploy.

### Environment Variables (optional)

No environment variables required for basic deployment.

## Audio Files

Place call recording audio files in `public/audio/`:
- `dentist.wav`
- `dealership.wav`
- `hardware.wav`
